Integrates MotoPress Page Builder with Zerif theme. Adds Zerif custom styles to all content elements added via powerful MotoPress Page Builder.

== Description ==

This plugin integrates your Zerif theme with a popular MotoPress Page builder to give you more freedom of building custom pages and posts keeping the initial theme styles. MotoPress editor allows you to add versatile in-demand content elements right on the frontend and this plugin automatically adjusts them to Zerif theme styles. 

This integration allows you to significantly simplify and functionally enrich your overall Zerif theme customization experience without affecting natural theme design.  

== Installation ==

1. Install the plugin via FTP or from Dashboard
2. Activate plugin
3. Make sure MotoPress Page Builder plugin is activated

Activate the plugin from your admin panel.
